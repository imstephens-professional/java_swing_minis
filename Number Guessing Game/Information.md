## Description
A Java typing game. The player has 30 seconds to guess a random number. The code informs the player whether they guessed significantly less, less, significantly higher, higher, or the exact number. There is a counter that keeps track of how many guesses the player got correct. 

Software developed in Eclipse Studio IDE (2060-03).
