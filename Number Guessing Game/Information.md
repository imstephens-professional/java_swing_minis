## Description
A Java typing game. The player must guess a random number. The code informs the player whether they guessed significantly lower, lower, significantly higher, higher, or the exact number. The game ends when the player successfully guesses the number or the player decides to give up.

Software developed in Eclipse Studio IDE (2026-03). Icons created in Clip Studio Paint (5.0.4).

This project is currently **complete**.
